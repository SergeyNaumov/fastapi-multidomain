Каждый сайт базируется на проекте и шаблоне. В папке проекта находятся контроллеры, отвечающие за бизнес-логику, тогда как а в папке шаблона только представление: шаблоны на основе jinja2, картинки, js, css

В папке с обязательно должно находится:

  • Роутер проекта (файл route.py)
  • Описание моделей (файл model_list.py)

Вот пример файла route.py (основного роутера): 

{% include '/page/main/pre_route.py' %}

Обратите внимание, что для удобства чтения route.py может включать в себя дочерние роутеры (в нашем примере это form_routes)
